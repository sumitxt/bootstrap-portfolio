PLC Ladder Diagram

I can design PLC diagram using RSLogix Micro. Ladder Diagram is a graphical programming language that is used for developing software for programmable logic controllers (PLCs). It is one of the languages that the IEC 61131 standard specifies for use with PLCs. A program in ladder diagram notation is a circuit diagram that emulates circuits of relay logic hardware.
